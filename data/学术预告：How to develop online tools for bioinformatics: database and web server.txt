How to develop online
tools for bioinformatics: database and web server
 王翊  博士
2017年3月30日（周四）16:00
逸夫楼C座314会议室
随着各种生物数据的不断累积，开发生物信息软件工具，特别是在线数据库和分析平台，进而方便研究人员使用，获得可视化的结果，是生物信息学研究的重要方向。本报告将分别以PIECE（植物基因内含子和外显子进化数据库）和OrthoVenn（同源蛋白在线聚类分析平台）为例子，按照软件的开发流程，讲述如何整合和利用生物信息数据来开发在线工具，分享开发过程中的技巧和经验。
王翊，男，博士，2002获浙江大学学士学位，2005年获电子科技大学硕士学位，2009年获重庆大学博士学位，随后在加州大学戴维斯分校和美国农业部西部研究中心进行博士后研究。现为加州大学伯克利分校Associate specialist，以生物信息专家的身份加入多个美国国家科学基金项目和美国农业部项目，进行数据处理和工具开发。研究方向主要为生物大数据分析，数据库和web开发，计算生物学，微生物组学等，在生物信息学，植物学和基因组学等相关SCI杂志上发表论文30余篇。
